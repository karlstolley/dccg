More Practice

Ok, so how can you know if you really understand it? Try moving some of the elements around to rearrange the infrastructure of the webpage. Please note that some of the elements cannot and should not be moved around (e.g. head, body, main navigation, etc.) If you are unsure about why, review the sections on Structure and Layout.

How about if you moved the table above the image? Or the bulleted list below the table? Hint: Remember the rules of bookends and nesting.

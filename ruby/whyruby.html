Why Ruby?

There are many programming languages available to a new programmer. Some you may know: Java, C and Javascript. Some you may not know: Haskell, Lisp or SmallTalk.

Ruby is flexible, popular, easy to code and free.

Flexible

Ruby can be used for most anything: web applications, system utilities or even games. It is not made to only do one thing.

Any programmer can also alter Ruby. If needed functionality does not exist in Ruby, the programmer can add it.

Ruby can be run from the command line, from a script or as a web application.

Popular

Ruby is one of the top ten most used programming languages. Ruby on Rails, introduced in 2003, allows for rapid development of web applications. Many well known sites use Rails: Groupon, LinkedIn, GitHub, Hulu, Basecamp, Penny Arcade and others.

Much of Ruby's popularity stems from it being easy to read and easy to code.

Easy to Read and Code

Ruby is written to appear as close to English as possible. It does not add more words or syntax to do something simple.

If you want to have your computer show "Hello world!" on the screen, you would need to do the following...

In Java

public class HelloWorld {
  public static void main(String[] args) {
    System.out.println("Hello world!");
  }
}

Translated, this says:

  1. Create a block of code that anyone can use to show "Hello world!".
  2. In this block, there is a block of code, a function, that runs by default, called main, whenever someone tries to use HelloWorld.
  3. This function does not require additional information.
  4. This function runs a command, called a method.
  5. This method prints a line of text ("Hello world!") out to the system (System.out.println).

Whew.

In Ruby

puts "Hello world!"

This says put "Hello world!" on the screen.

Which language would you rather learn?

Free

Ruby can be downloaded and installed at no cost. All you need to program with it is a text editor; no specialized programs (called Integrated Development Environments or IDEs) are required.

Accessing the REPL

By the End Of This Section You Should Be Able To...

  • Describe what a REPL is
  • Access the REPL in your favorite browser

Below is a list of the common popular web browsers and instructions on how to access what is called either their JavaScript console or JavaScript REPL. You'll be using this both in the rest of this section as well as the rest of the JavaScript chapter.

Google Chrome

To open up the JavaScript console on Windows and Linux in Chrome, press Ctrl-Shift-J. In Mac OS, press Command-Option-J. You should see something like this:

Firefox

To open the JavaScript console on Windows and Linux in Firefox, press Ctrl-Shift-K. In Mac OS, press Command-Option-K. You should see something like this:

The default Firefox console should work fine for going through the JavaScript lessons here, but a popular alternative to the default console is Firebug, which is available as a Firefox add-on that has a lot of other neat features.

Internet Explorer

To open the JavaScript console in Internet Explorer, press the F12 button. This will bring up the Internet Explorer Developer Tools. Once this is open, click on the "Script" tab to access the JavaScript console. You should see something like this:

Safari

To open the JavaScript console in Safari, first you have to enable it. Open the Safari preferences and go to the "Advanced" tab. Then click the "Show Develop menu in menu bar" checkbox.

What is a REPL?

Let's get some practice with these things inside of our JavaScript REPL. I've been referring to this thing we've been looking for as a REPL, but what does that mean? It stands for Read Eval Print Loop. At first, this might look like jargon, but let's break it down.

When you type something into the REPL, first it reads it. That just means it looks at what you typed in. Then it evaluates it. We'll get into this in the next section, but what this basically means is it applies some rules to what you typed in. Finally, it prints the results. Loop just means it never stops -- after it prints out the result, it will be ready to read in the next thing you have to say.

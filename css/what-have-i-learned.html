What Have I Learned?

If you have completed the CSS chapter, you should now be familiar with the following topics:

CSS Selectors

Selectors are the syntax that allow you to identify which HTML5 objects you want to target for your CSS rules and apply declarations that will change the look and feel of your page.

Return to the Selectors section.

CSS Declarations

Declarations are applied to CSS selectors within a web page. They consist of two parts: a property and a corresponding value.

Return to the Declarations section.

Using CSS Rules for Positioning

Positioning refers to creating CSS rules to places elements in a particular area on the page, regardless of where in the document it is references.

Return to the Positioning section.

Using Classes and Ids in CSS Rules

Classes and IDs are applied to HTML5 elements to distinguish them from similar elements. When you want to apply the same style rules to a specific set of elements, add a class name to those elements. If you want to apply style rules to only one element, refer to it by a unique ID name.

Return to the Classes and IDs as Selectors section.

Storing CSS Rules in an External Stylesheet

Maintaining your style rules is much easier if you store them in external stylesheets. This keeps you from having to find the specific HTML file that you have have put the rule in and lets you make changes much more quickly.

Return to the Stylesheets section.

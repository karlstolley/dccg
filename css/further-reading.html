Further Reading

Want more CSS? Check out these sources:

  • A List Apart

    Industry-leading blog that discusses best practices of web development and showcases new and upcoming specifications

  • Code Academy

    Online tutorials in many programming languages, including a section on HTML and CSS.

  • Sitepoint CSS Reference

    Unlike some other reference sites that are outdated and poorly-written but highly ranked, Sitepoint is a reliable online reference source.

  • CSS3 for Web Designers

    From the publication arm of A List Apart, an in-depth look at all that CSS3 has to offer.

  • CSS3 Generator

    A fun tool that lets you explore many of the popular new rules of the CSS3 specification.

  • Google Web Fonts API

    A application programming interface (API) from Google that will allow you generate hundreds of different fonts to use with CSS3 rules.
